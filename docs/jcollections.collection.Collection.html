class Collection(Iterable):

Known subclasses: jcollections.abstractqueue.AbstractQueue, jcollections.arraydeque.ArrayDeque, jcollections.list.List, jcollections.priorityqueue.PriorityQueue, jcollections.set.Set

Abstract class representing a general collection of elements. Extends the Iterable class to allow for iteration over the collection. Subclasses must implement all abstract methods to define specific collection behaviors.
