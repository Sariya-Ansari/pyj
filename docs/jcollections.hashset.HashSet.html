jcollections.hashset.HashSet

class documentation

class HashSet(Set):

Constructor: HashSet(initial_capacity, load_factor, elements)

View In Hierarchy

HashSet implementation that does not allow duplicate elements and does not guarantee any specific order of elements.

This class uses Python's built-in set as the internal storage. The initial capacity and load factor are ignored since Python sets handle these aspects automatically.
