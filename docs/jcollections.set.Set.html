jcollections.set.Set

class documentation

class Set(Collection, Iterable, ABC):

Known subclasses: jcollections.hashset.HashSet, jcollections.linkedhashset.LinkedHashSet, jcollections.sortedset.SortedSet

View In Hierarchy

Abstract base class for a Set.

A Set is a collection that does not allow duplicate elements and does not guarantee any specific order of elements. This class inherits from Collection and Iterable, making it both a Collection and an Iterable.

Inherits from:

  • Collection: Provides methods for adding, removing, and querying elements.
  • Iterable: Provides methods for iterating over elements.
  • ABC: Abstract Base Class to enforce that this is an abstract class.

Creates a Spliterator over the elements in this set (optional operation).

A Spliterator is used to traverse and partition elements of the Set. This method is optional and can be used to provide a custom implementation if needed. By default, it is not implemented in Python.
