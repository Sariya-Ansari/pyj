jcollections

package documentation

The `jcollections` package provides various collection classes such as `ArrayList`, `HashMap`, and `HashSet` etc. These classes mimic similar structures from the Java Collections Framework, offering robust data structures for handling collections in Python.

Module arraylist The `arraylist` module implements the `ArrayList` class, a dynamic array-backed list that resizes itself as elements are added. It provides similar functionality to Java's `ArrayList`.
Module collection The `collection` module defines the `Collection` abstract class, which serves as the foundation for various collection types. It includes common methods such as `add`, `remove`, and `contains`.
Module hashmap The `hashmap` module implements the `HashMap` class, a key-value pair data structure that provides fast lookups, insertions, and deletions, similar to Java's `HashMap`.
Module hashset The `hashset` module implements the `HashSet` class, which is a set implementation backed by a `HashMap`, ensuring unique elements without any specific order.
Module hashtable This module defines a thread-safe implementation of a Hashtable that uses locks to ensure concurrency. The Hashtable class supports basic key-value storage, retrieval, and map operations.

Module linkedhashset This module provides a `LinkedHashSet` class, which extends the `Set` class to maintain the insertion order of elements while ensuring that elements are unique.

Module mapentry This module defines the `MapEntry` class, which represents a key-value pair in a map-like data structure. It allows for the retrieval, modification, and comparison of key-value pairs.
Module Predicate This module defines the `Predicate` class, which is used to encapsulate a function that evaluates a condition and can be applied to various elements. It allows for the creation of predicates that can be reused and applied to any item.
Module priorityqueue This module defines a `PriorityQueue` class, which provides a priority-based ordering of elements using a heap. The class supports common queue operations such as adding, peeking, polling, and more.
Module queue This module defines a `Queue` class that provides a basic queue implementation using either an `ArrayList` or a `LinkedList` for internal storage.
Module set This module defines an abstract base class `Set`, which represents a collection of unique elements.
Module sortedmap This module defines an abstract base class `SortedMap`, which represents a sorted map of key-value pairs.
Module sortedset This module defines an abstract base class `SortedSet`, which represents a sorted set of unique elements.
Module stack This module defines the `Stack` class, which is a last-in-first-out (LIFO) stack of elements. The `Stack` class inherits from the `Vector` class and provides additional methods for stack operations such as push, pop, peek, and search.

Module treeset This module defines the `TreeSet` class, a sorted set implementation that maintains elements in ascending order.
