class ArrayDeque(Collection):

Constructor: ArrayDeque(initial)

View In Hierarchy

ArrayDeque is a resizable-array implementation of the Deque (double-ended queue) interface. It supports element insertion and removal from both ends of the queue. The class offers operations for efficiently adding elements at both the front and back, while also allowing for element removal and access. Inherits from: Collection: The base interface for all collection types. Attributes: _deque (list): The underlying list that stores elements. _capacity (int, optional): The initial capacity of the deque if specified.
