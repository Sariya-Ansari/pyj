jcollections.priorityqueue.PriorityQueue

class documentation

class PriorityQueue(Collection, Queue):

Constructor: PriorityQueue(initial, comparator, initial_capacity)

View In Hierarchy

A priority queue implementation using a binary heap. This class allows elements to be stored and retrieved based on their priority. An optional comparator can be provided for custom ordering. Attributes: _heap (list): Internal heap structure to store elements. comparator (Callable, optional): A function to compare elements for priority ordering.
