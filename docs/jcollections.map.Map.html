jcollections.map.Map

class documentation

class Map(ABC, Generic[K, V]):

Known subclasses: jcollections.hashmap.HashMap, jcollections.hashtable.Hashtable

View In Hierarchy

An abstract base class defining a map (dictionary) data structure. This class outlines the fundamental operations that any concrete map implementation must support. Type Parameters: K: Type of the keys in the map. V: Type of the values in the map.
